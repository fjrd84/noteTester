    NoteTester.js

    NoteTester.js is a responsive on-screen piano-keyboard entirely designed using HTML, CSS and Javascript.


    It has been developed for the Cassette Coach e-learning music page.

    If you want to use it in your page, you just need to:

    • Include the whole contents of the .noteTesterWrapper div in your DOM. You can find it in the noteTester.html file of the repository.
    • Then import noteTester.css and noteTester.js, and that's all!

    You can define its size by modifying the css definition of the class "noteTestWrapper".

    For more information, check out the noteTester github repository.
